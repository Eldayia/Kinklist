# Propositions de design

Ces fichiers sont des maquettes autonomes. Ils ne sont ni chargés par le site actuel ni exposés par son serveur Express.

## Ouvrir les maquettes

Ouvrir directement `index.html` dans un navigateur, puis choisir une proposition.

Pour les consulter via un serveur de développement indépendant :

```powershell
cd design-previews
python -m http.server 8080
```

Puis ouvrir `http://localhost:8080`.

## Concepts

- `editorial.html` : direction claire, chaleureuse et rassurante, proche d'un carnet personnel.
- `cockpit.html` : direction compacte et fonctionnelle, adaptée au volume important de kinks.
- `nocturne.html` : direction sombre, intime et plus expressive.

Les trois variantes conservent les formes géométriques des statuts, les rôles Donne/Reçoit, les définitions, la recherche et les catégories repliables. Les données affichées sont uniquement des exemples.
